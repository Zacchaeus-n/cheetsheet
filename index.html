HTML Table Reference

Block-level Elements

Tag Name Description
<p> Paragraph Element contains and displays a block of text.
<ul> Unordered List Element unordered list element is used to create a list of items in no particular order.
<li> List Item Element list item element create list items inside
<ol> Ordered List Element ordered list element creates a list of items in sequential order.

Inline Elements

Attribute Name Description
<span> Span Element an inline container for text and can be used to group text for styling purposes.
<em> Emphasis Element emphasizes text and browsers will usually italicize the emphasized text by default.
<strong> Strong Element Defines important text
